# 5·17 성소수자 평등의 날 공동행동 평등위원 프로필 꾸미기

프로필 사진에 아이템을 추가해 성소수자 '평등 실현'과 '권리 보장'을 요구해주세요!

[rainbowaction-517.vercel.app](https://rainbowaction-517.vercel.app)

## 주요 기능

- 프로필 사진 업로드 또는 촬영
- 아이템(스티커) 선택 및 위치·크기·회전 조절
- 완성된 이미지 다운로드 및 공유
